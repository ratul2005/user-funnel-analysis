# 📊 User Funnel Analysis

## 📌 Problem Statement

In an e-commerce platform, many users visit but do not complete purchases.
This project analyzes the user journey to identify where users drop off and how to improve conversion.

---

## 📁 Dataset

* Source: Simulated / Kaggle E-commerce dataset
* Contains user interaction stages:

  * homepage
  * product_page
  * cart
  * checkout
  * purchase

---

## 🧠 Objectives

* Analyze user flow across funnel stages
* Calculate conversion rate at each step
* Identify major drop-off points
* Suggest improvements to increase conversions

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib

---

## 📊 Funnel Stages

Homepage → Product Page → Cart → Checkout → Purchase

---

## 📈 Analysis

### Stage-wise Users

* Homepage: 10,000
* Product Page: 5,000
* Cart: 1,500
* Checkout: 450
* Purchase: 225

---

### Conversion Rates

* Homepage → Product Page: 50%
* Product Page → Cart: 30%
* Cart → Checkout: 30%
* Checkout → Purchase: 50%

---

## 💡 Key Insights

* Major drop-off occurs from **Product Page → Cart (70% drop)**
* Many users browse but do not add products to cart
* Checkout stage retains users relatively well

---

## 🚀 Recommendations

* Improve product page design (better CTA, trust signals)
* Add discounts or urgency (limited offers)
* Simplify add-to-cart process
* Provide better product information

---

## 📌 Conclusion

User drop-off is highest at the product exploration stage.
Improving engagement and trust at this step can significantly increase overall conversion.
